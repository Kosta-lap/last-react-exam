# Inventory Manager

Минимальное SPA-приложение для управления складом музыкального магазина.

## Возможности

- Просмотр статистики по складу (общее количество, самые дорогие и самые дешевые товары).
- Добавление новых товаров и пополнение существующих позиций.
- Удаление товара из списка.
- Продажа товара с проверкой доступного количества.
- Автоматический пересчет баланса после закупки и продажи.
- История операций с датой и описанием.

## Технологии

- React
- React Router
- Redux Toolkit

## Запуск

```bash
npm install
npm start
```
